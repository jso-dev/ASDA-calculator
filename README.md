# ASDA-calculator
Build a calculator app for ASDA

## Instructions for use
Assuming you have Node and Git installed on your machine (if not, see Installing Node and Git below); using the bash terminal of your IDE navigate to the directory you would like to clone the repository to and type:
    'git clone https://github.com/jso-dev/ASDA-calculator.git'

Once you have cloned the repository, open the directory in your IDE and using the terminal type 'npm install' to install all dependencies.

Once all dependencies have been installed type 'npm start'. This will launch the app on localhost:3000 in your default browser.

## Installing Node and Git
If you do not have Node or Git installed use the following links to install them onto your machine. Once done, follow the instructions above:

Latest version of Node: https://nodejs.org/en/download/

Latest version of Git: https://git-scm.com/downloads

## Instructions for testing
Once you have completed the initial instructions for use; using your IDE terminal run 'npm test' to run the testing suite.

## Limitations
While through equations you can create a negative number the calculator does not allow you to enter a negative number.
